# go-minidump
   ![Demo](./icons/demo.png)

## Description

tool for dump process full memory(default lsass.exe)

## Getting Started
```
make build
```
then, binary in /build.
## Usage
```
Usage: minidump.exe -f <file name> -p <process id> -n <process name>
```
